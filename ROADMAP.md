# ClassFinal 功能规划

> 本文档记录 ClassFinal 的功能规划和开发方向，基于实际需求和可行性制定。

## 🚀 v2.0.1 (开发中)

**核心目标**：改善易用性和安全性

### 配置文件支持
- [ ] 支持 YAML/JSON 配置文件 (`classfinal.yml`)
- [ ] 生成配置模板 `--init-config`
- [ ] 环境变量占位符 `${VAR_NAME}`
- [ ] 配置文件验证和友好错误提示

### 密码管理增强
- [ ] 从文件读取密码 `--password-file /path/to/pwd.txt`
- [ ] 密码文件自动删除选项（读取后删除）
- [ ] 改进环境变量密码读取逻辑
- [ ] 密码强度检查和提示

### 加密验证工具
- [ ] 验证 JAR 是否已加密 `--verify app.jar`
- [ ] 显示加密统计（加密类数量、加密率等）
- [ ] 检测加密方法和保护模式
- [ ] 列出已加密的类名列表

### 用户体验改进
- [ ] 命令行参数标准化（支持 `--long-option`）
- [ ] 更友好的错误提示（中英文）
- [ ] 日志级别控制 `--log-level`
- [ ] 进度条显示优化

### 文档完善
- [ ] 配置文件使用指南
- [ ] 密码管理最佳实践
- [ ] 常见问题 FAQ 扩展
- [ ] 安全加固建议

---

## 🎯 v2.1.0 (规划中)

**核心目标**：支持批量处理和性能优化

### 批量加密
- [ ] 批量配置文件 `--batch jobs.yml`
- [ ] 一次加密多个 JAR/WAR
- [ ] 批量加密进度报告
- [ ] 失败重试机制

### 性能优化
- [ ] 多线程并行加密 `--threads N`
- [ ] 大文件流式处理
- [ ] 增量加密支持 `--incremental`（只加密变更的类）
- [ ] 加密缓存机制

### 高级匹配
- [ ] 包名通配符 `com.example.**`
- [ ] 正则表达式匹配
- [ ] 按注解选择 `@Encrypted`

### 配置文件加密
- [ ] application.yml 字段级加密
- [ ] properties 文件敏感值加密
- [ ] 运行时自动解密
- [ ] 加密标记语法 `ENC(...)`
